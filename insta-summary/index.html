AIで「要約」もしたい場合は？

全文翻訳ではなく、AIによる要約（要点を数行にまとめる）には、Claude などのAIの API キーが必要です（有料・従量課金）。 無料で使いたい場合は、このページの翻訳機能をご利用ください。AI要約版に切り替えたいときは教えてください。
